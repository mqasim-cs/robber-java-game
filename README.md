# Robber - Java 2D Game

A 2D arcade-style game built with Java Swing where the player controls a thief collecting coins while escaping police enemies.

## Features
- Animated start screen
- Collision detection
- Enemy AI movement
- Score and lives system
- Game Over and Win screens
- Keyboard controls
- Coin collection mechanics

## Technologies
- Java
- Java Swing
- OOP
- IntelliJ IDEA

## Controls
↑ ↓ ← → Arrow Keys
